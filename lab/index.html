Simple Calculator Project

Create a calculator using HTML, CSS, and JavaScript. Your code must meet these requirements:

  • HTML: Include elements with IDs: num1 (input), num2 (input), operation (select), calculate-button, result (div).
  • CSS: Style inputs, select, button, and result for a clean, responsive layout.
  • JavaScript:
    • Define a calculate function to perform the selected operation (+, -, *, /).
    • Handle button click to compute and display the result.
    • Check for invalid inputs (e.g., empty fields, division by zero).
    • Update the result element with the answer or error message.

Write your code in the tabs below. The output updates as you type. Click "Run Tests" to validate your code.

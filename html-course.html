1. Basic Structure of an HTML Page
<!DOCTYPE html><html><head><title>My Page</title></head><body></body></html>

Every HTML page starts with a basic structure:

  • <!DOCTYPE html>: Declares the document as HTML5.
  • <html>: The root element containing all content.
  • <head>: Holds metadata like the title.
  • <body>: Contains visible content.

Try it: Save this as index.html and open it in a browser!

2. Headings in HTML
<h1>Heading 1</h1><h2>Heading 2</h2>

Headings organize content by importance:

  • <h1> to <h6>: Six levels, <h1> is the largest and most important.

Try it: Add these to your <body> and see the size difference!

3. Paragraphs in HTML
<p>This is text.</p>

Paragraphs display blocks of text:

  • <p>: Creates a paragraph with automatic spacing.

Try it: Add this to your <body> to see text!

4. Links in HTML
<a href="https://example.com">Click</a>

Links connect pages or sites:

  • <a>: Anchor tag, with href setting the destination URL.

Try it: Add this and click to visit the URL!

5. Images in HTML
<img src="image.jpg" alt="Description">

Images add visuals:

  • <img>: Displays an image.
  • src: Path to the image file.
  • alt: Text description for accessibility.

Try it: Use an image file path and see it appear!

6. Comments in HTML
<!-- This is a comment -->

Comments are notes in code:

  • <!-- -->: Text inside is hidden from the page but visible in code.

Try it: Add a comment and check it doesn’t show!

7. Unordered Lists
<ul><li>Item</li></ul>

Unordered lists use bullets:

  • <ul>: Starts a bullet list.
  • <li>: Each list item.

Try it: Add a list and see bullets!

8. Ordered Lists
<ol><li>Step</li></ol>

Ordered lists use numbers:

  • <ol>: Starts a numbered list.
  • <li>: Each list item.

Try it: Add a numbered list!

9. Divs in HTML
<div>Content</div>

Divs group content:

  • <div>: A block container for organizing elements.

Try it: Wrap some content in a div!

10. Spans in HTML
<span>Text</span>

Spans style inline text:

  • <span>: Inline container, doesn’t break lines.

Try it: Add a span in a paragraph!

11. Line Breaks
<br>

Line breaks split text:

  • <br>: Adds a single line break without extra spacing.

Try it: Use it between two lines!

12. Horizontal Rules
<hr>

Horizontal rules divide sections:

  • <hr>: Draws a horizontal line.

Try it: Add it between content!

13. Bold and Italic Text
<b>Bold</b> <i>Italic</i>

Format text for emphasis:

  • <b>: Makes text bold.
  • <i>: Makes text italic.

Try it: Format some text!

14. Superscript and Subscript
<sup>2</sup> <sub>2</sub>

Position text above or below:

  • <sup>: Superscript (e.g., exponents).
  • <sub>: Subscript (e.g., formulas).

Try it: Write 22 or H2O!

15. Blockquotes
<blockquote>Quote</blockquote>

Blockquotes highlight quotes:

  • <blockquote>: Indents and styles a quotation.

Try it: Add a quote!

16. Meta Tags
<meta charset="UTF-8">

Meta tags add page info:

  • <meta>: Sets metadata, like character encoding.

Try it: Add to <head>!

17. Tables
<table><tr><td>Cell</td></tr></table>

Tables organize data:

  • <table>: Creates a table.
  • <tr>: Table row.
  • <td>: Table cell.

Try it: Build a small table!

18. Table Headers
<th>Header</th>

Table headers label data:

  • <th>: Bold, centered header cell.

Try it: Add headers to a table!

19. Forms
<form></form>

Forms collect user input:

  • <form>: Container for input elements.

Try it: Start a form!

20. Input Fields
<input type="text">

Inputs gather data:

  • <input>: Creates an input field.
  • type="text": For text entry.

Try it: Add to a form!

21. Buttons
<button>Click</button>

Buttons trigger actions:

  • <button>: Creates a clickable button.

Try it: Add a button!

22. Dropdowns
<select><option>Choice</option></select>

Dropdowns offer choices:

  • <select>: Creates a dropdown.
  • <option>: Each choice.

Try it: Add a dropdown!

23. Textareas
<textarea></textarea>

Textareas allow multi-line text:

  • <textarea>: For longer input.

Try it: Add to a form!

24. Labels
<label for="id">Text</label>

Labels describe inputs:

  • <label>: Links to an input via for.

Try it: Label an input!

25. Fieldsets
<fieldset><legend>Title</legend></fieldset>

Fieldsets group form elements:

  • <fieldset>: Groups inputs.
  • <legend>: Titles the group.

Try it: Group form elements!

26. Iframes
<iframe src="page.html"></iframe>

Iframes embed content:

  • <iframe>: Displays another page inside yours.

Try it: Embed a URL!

27. Audio
<audio src="sound.mp3" controls></audio>

Audio adds sound:

  • <audio>: Plays audio files.
  • controls: Adds playback buttons.

Try it: Add an audio file!

28. Video
<video src="video.mp4" controls></video>

Video embeds movies:

  • <video>: Plays video files.
  • controls: Adds playback controls.

Try it: Add a video!

29. Semantic Tags
<header><nav><section><footer>

Semantic tags add meaning:

  • <header>: Top section.
  • <nav>: Navigation.
  • <section>: Content section.
  • <footer>: Bottom section.

Try it: Structure your page!

30. Data Attributes
<div data-info="value"></div>

Data attributes store info:

  • data-*: Custom attributes for extra data.

CSS Basics and Styling

Welcome to Lesson 10 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn CSS basics to style HTML pages. By the end, you’ll be able to apply colors, fonts, and layouts to create visually appealing web pages.

Introduction

CSS is like a shepherd’s dye, transforming plain wool (HTML) into vibrant, styled pages. It controls the look and feel of a website, from colors to fonts. This lesson introduces CSS selectors, properties, and methods to style your HTML content effectively.

Key Concepts

This lesson covers CSS fundamentals:

  • Selectors: Target elements using tags (e.g., h1), classes (.class), or IDs (#id).
  • Properties: Style elements with properties like color, font-size, and background.
  • CSS Methods: Apply styles via inline, internal (in <style>), or external (separate .css file) methods.

Code Example

Below is an example of styling a semantic HTML page with an external CSS file.

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Styled Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1 class="title">Welcome</h1>
    <p>This is a styled paragraph.</p>
</body>
</html>

<!-- styles.css -->
.title {
    color: blue;
    font-size: 24px;
}
p {
    background: lightgray;
    padding: 10px;
}

Practical Applications

CSS is essential for creating professional, user-friendly websites, like those of Amazon or The New York Times. For example, in a blog app, CSS styles headers to grab attention and paragraphs to ensure readability. Mastering CSS allows you to enhance user experience in full-stack applications.

Tips for Success

Here are tips to master CSS styling:

  • Use external stylesheets: Keep CSS in a separate file for maintainability.
  • Test styles in dev tools: Use Chrome’s inspector to experiment with properties.
  • Keep it simple: Avoid overly complex selectors to maintain clean code.

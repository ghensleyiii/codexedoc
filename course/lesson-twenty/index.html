JavaScript Variables and Data Types

Welcome to Lesson 20 of CODEXEDOC’s Full-Stack Development Course! Variables and data types are the building blocks of JavaScript programming. In this lesson, you’ll learn how to store data using variables and understand the different data types JavaScript supports, enabling you to manipulate information effectively.

What are Variables?

Variables are containers for storing data, like numbers or text, that can be used and modified in your code. JavaScript uses keywords like let, const, and var to declare variables. In 2025, let and const are preferred for modern coding.

Why Use Variables?

Variables allow you to store user input, calculations, or state, making your code dynamic. For example, you can store a user’s name or a product’s price and update them as needed. Understanding variables is crucial for writing flexible JavaScript programs.

Key Data Types

JavaScript supports these main data types:

  • Number: Integers or decimals (e.g., 42, 3.14).
  • String: Text enclosed in quotes (e.g., "Hello").
  • Boolean: True or false values (true, false).
  • Object: Complex data structures (e.g., { name: "John" }).
  • Undefined: A variable with no value (undefined).

Working with Variables

Follow these steps to use variables:

  1. Declare Variables: Use let or const (e.g., let age = 25;).
  2. Assign Values: Update variables with new data (e.g., age = 26;).
  3. Check Types: Use typeof to inspect a variable’s data type.

Code Example: Variables and Data Types

Here’s a page demonstrating variables and data types:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Variables Page</title>
</head>
<body>
    <h1>User Info</h1>
    <p id="output"></p>
    <script>
        let name = "Alice";
        const age = 30;
        let isStudent = true;
        let person = { name: name, age: age };
        document.getElementById("output").textContent = 
            `Name: ${name}, Age: ${age}, Student: ${isStudent}, Type: ${typeof person}`;
        console.log("Person object:", person);
    </script>
</body>
</html>

This example declares variables with different data types, displays them on the page, and logs an object to the console.

Practical Applications

Variables store data for calculations, user input, or state management. For example, an e-commerce site uses variables to track cart items or prices. In this course, you’ll use variables to manage product quantities in your e-commerce project.

Tips for Success

To work with variables:

  • Use const for Constants: Prefer const for values that won’t change.
  • Choose Clear Names: Use descriptive names like userName for readability.
  • Check Types: Use typeof to debug unexpected values.
  • Avoid var: Use let or const for modern scoping.

Summary

Variables and data types are essential for storing and manipulating data in JavaScript. By mastering let, const, and common data types, you’re ready to build dynamic features. In the next lesson, you’ll learn about functions and scope to organize your code. Keep practicing variables!

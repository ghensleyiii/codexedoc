Git Basics

Welcome to Lesson 5 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn the core Git commands to track changes in your code. By the end, you’ll be able to initialize a repository, stage files, and commit changes, setting the foundation for version control.

Introduction

Git is like a shepherd tracking a flock’s history, ensuring every change to your code is saved and reversible. As a version control system, Git lets you save snapshots of your project, making it easy to experiment, revert mistakes, or collaborate with others. This lesson introduces the basic commands to manage your code effectively.

Key Concepts

This lesson covers the fundamentals of Git:

  • Repository: A project folder tracked by Git, containing your code and its history.
  • git init: Initializes a new Git repository in your project folder.
  • git add: Stages changes for a commit.
  • git commit: Saves staged changes with a message, creating a snapshot.

Code Example

Below is an example of using Git to track an HTML file. These commands create a repository, add a file, and commit it.

# Initialize a Git repository
git init

# Create an HTML file
touch index.html

# Stage the file
git add index.html

# Commit the file with a message
git commit -m "Add initial HTML file"

Practical Applications

Git is essential for full-stack development, used by teams at companies like GitHub and Atlassian to manage codebases. For example, in a web app project, Git tracks changes to HTML, CSS, and JavaScript files, allowing you to revert errors or collaborate seamlessly. Mastering Git basics ensures you can manage your code like a pro.

Tips for Success

Here are tips to master Git basics:

  • Commit often: Save small, logical changes with clear messages.
  • Check status: Use `git status` to see what’s staged or modified.
  • Practice locally: Create a test repository to experiment without risk.

Shearing Tip

Git shears away chaos by saving your code’s history, just as a shepherd keeps a flock organized. With Git, you can track changes and revert mistakes with ease.

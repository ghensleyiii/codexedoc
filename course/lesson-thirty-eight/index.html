Django REST Framework

Mastering Django REST Framework

Welcome to Lesson 38! Django REST Framework (DRF) extends Django for building REST APIs. In this lesson, you’ll learn to create APIs with DRF, including serializers, viewsets, and authentication.

What is Django REST Framework?

DRF, built on Django, simplifies REST API development with tools for serialization and authentication. In 2025, it’s a standard for Python APIs, akin to Express for Node.js.

Key DRF Concepts

Focus on these features:

  • Serializers: Convert models to JSON.
  • Viewsets: Handle CRUD operations.
  • Authentication: Use JWT or token-based auth.
  • Permissions: Restrict access to endpoints.

Code Example: DRF API

Here’s a DRF API for posts:

# models.py
from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()

# serializers.py
from rest_framework import serializers
from .models import Post

class PostSerializer(serializers.ModelSerializer):
    class Meta:
        model = Post
        fields = ['id', 'title', 'content']

# views.py
from rest_framework import viewsets
from .models import Post
from .serializers import PostSerializer

class PostViewSet(viewsets.ModelViewSet):
    queryset = Post.objects.all()
    serializer_class = PostSerializer

# urls.py
from django.urls import path, include
from rest_framework.routers import DefaultRouter
from . import views

router = DefaultRouter()
router.register(r'posts', views.PostViewSet)

urlpatterns = [
    path('api/', include(router.urls)),
]

Run python manage.py runserver to access localhost:8000/api/posts/.

Practical Applications

DRF powers APIs for apps like dashboards. In your social media dashboard project, you’ll use DRF for post and user APIs.

Tips for Success

To master DRF:

  • Install DRF: Use pip install djangorestframework.
  • Use Viewsets: Simplify CRUD operations.
  • Test APIs: Use Postman or DRF’s browsable API.
  • Add Authentication: Secure endpoints with JWT.

Summary

DRF streamlines REST API development. By mastering serializers and viewsets, you’re ready for API-driven apps. Next, you’ll learn database integration. Keep building with DRF!

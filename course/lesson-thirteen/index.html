Responsive Design with Media Queries

Welcome to Lesson 13 of CODEXEDOC’s Full-Stack Development Course! Responsive design ensures your websites look great on all devices, from desktops to smartphones. In this lesson, you’ll learn how to use CSS media queries to adapt layouts based on screen size, creating mobile-friendly designs.

What is Responsive Design?

Responsive design is a web development approach that makes websites adapt to different screen sizes and devices. Introduced with CSS3, it uses techniques like media queries and flexible units (e.g., vw, rem). In 2025, responsive design is critical, as mobile traffic accounts for over half of web visits.

Why Use Media Queries?

Media queries allow you to apply CSS styles conditionally based on criteria like screen width, height, or device type. They ensure your site is usable and visually appealing on phones, tablets, and desktops, improving user experience and SEO.

Key Media Query Concepts

Media queries use the @media rule to target specific conditions:

  • Breakpoint: A screen size where styles change (e.g., 768px for tablets).
  • Min-Width/Max-Width: Apply styles above (min-width) or below (max-width) a breakpoint.
  • Mobile-First: Start with base styles for smaller screens, then add styles for larger screens.

Writing Media Queries

Follow these steps to implement media queries:

  1. Define Base Styles: Write default styles for mobile devices.
  2. Add Media Queries: Use @media (min-width: 768px) to adjust for larger screens.
  3. Test Responsiveness: Use browser dev tools to simulate different screen sizes.

Code Example: Responsive Layout

Here’s a responsive page using media queries:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>My Site</h1>
    </header>
    <main class="container">
        <div class="card">Card 1</div>
        <div class="card">Card 2</div>
    </main>
</body>
</html>

<!-- styles.css -->
.container {
    display: flex;
    flex-direction: column;
    gap: 10px;
}
.card {
    background: #f0f0f0;
    padding: 10px;
}
@media (min-width: 768px) {
    .container {
        flex-direction: row;
    }
    .card {
        width: 50%;
    }
}

This example stacks cards vertically on mobile and arranges them side-by-side on screens wider than 768px.

Practical Applications

Responsive design is essential for modern websites. For example, an e-commerce site uses media queries to adjust product grids for mobile users. In this course, you’ll apply media queries to make your e-commerce product page responsive, ensuring a great user experience across devices.

Tips for Success

To create responsive designs:

  • Use Relative Units: Use rem, vw, or % for flexible sizing.
  • Test on Real Devices: Check your site on actual phones and tablets.
  • Plan Breakpoints: Choose breakpoints based on content, not arbitrary device sizes.
  • Optimize Images: Use responsive images with srcset to reduce load times.

Summary

Media queries enable responsive designs that adapt to any device, improving usability and SEO. By mastering responsive techniques, you’re ready to create mobile-friendly websites. In the next lesson, you’ll explore advanced CSS techniques to enhance your designs further. Keep testing your responsive layouts!

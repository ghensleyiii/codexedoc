State Management with Redux

Welcome to Lesson 43 of CODEXEDOC’s Full-Stack Development Course! Redux manages complex state in React apps. In this lesson, you’ll add Redux to your e-commerce frontend, centralizing cart and user state for better scalability.

What is Redux?

Redux is a state management library that provides a predictable state container. In 2025, it’s widely used for managing global state in React apps.

Why Use Redux?

Redux simplifies state management for large apps, like your e-commerce store, by centralizing cart and authentication state, reducing prop drilling.

Key Redux Concepts

Focus on these techniques:

  • Store: Single source of truth for state.
  • Actions: Payloads describing state changes.
  • Reducers: Functions updating state based on actions.
  • Toolkit: Use @reduxjs/toolkit for simplicity.

Setting Up Redux

Follow these steps:

  1. Install Redux: Add @reduxjs/toolkit and react-redux.
  2. Create Slice: Define cart state and actions.
  3. Configure Store: Set up the Redux store.
  4. Connect Components: Use hooks to access state.

Code Example: Redux Cart

Here’s Redux integrated into Lesson 42’s frontend:

<!-- src/main.jsx -->
import React from 'react';
import ReactDOM from 'react-dom/client';
import { Provider } from 'react-redux';
import store from './store';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
    <React.StrictMode>
        <Provider store={store}>
            <App />
        </Provider>
    </React.StrictMode>
);

<!-- src/store.js -->
import { configureStore } from '@reduxjs/toolkit';
import cartReducer from './cartSlice';

export default configureStore({
    reducer: {
        cart: cartReducer
    }
});

<!-- src/cartSlice.js -->
import { createSlice } from '@reduxjs/toolkit';

const cartSlice = createSlice({
    name: 'cart',
    initialState: [],
    reducers: {
        addToCart(state, action) {
            const item = state.find(item => item.id === action.payload.id);
            if (item) {
                item.quantity += 1;
            } else {
                state.push({ ...action.payload, quantity: 1 });
            }
        },
        removeFromCart(state, action) {
            return state.filter(item => item.id !== action.payload);
        }
    }
});

export const { addToCart, removeFromCart } = cartSlice.actions;
export default cartSlice.reducer;

<!-- src/App.jsx -->
import { useSelector, useDispatch } from 'react-redux';
import { addToCart, removeFromCart } from './cartSlice';

function App() {
    const cart = useSelector(state => state.cart);
    const dispatch = useDispatch();
    const products = [
        { id: 1, name: 'Laptop', price: 999.99 }
    ];

    return (
        <div className="p-4">
            <header className="bg-blue-600 text-white p-4">
                <h1 className="text-2xl font-bold">Store</h1>
            </header>
            <main className="container mx-auto">
                <h2 className="text-xl font-bold my-4">Products</h2>
                <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
                    {products.map(product => (
                        <div key={product.id} className="bg-white p-4 rounded shadow">
                            <h3 className="font-bold">{product.name}</h3>
                            <p>${product.price.toFixed(2)}</p>
                            <button
                                className="bg-blue-600 text-white px-4 py-2 rounded"
                                onClick={() => dispatch(addToCart(product))}
                            >
                                Add to Cart
                            </button>
                        </div>
                    ))}
                </div>
                <h2 className="text-xl font-bold my-4">Cart</h2>
                <ul className="bg-white p-4 rounded shadow">
                    {cart.map(item => (
                        <li key={item.id} className="flex justify-between py-2">
                            <span>{item.name} (x{item.quantity}) - ${(item.price * item.quantity).toFixed(2)}</span>
                            <button
                                className="text-red-600"
                                onClick={() => dispatch(removeFromCart(item.id))}
                            >
                                Remove
                            </button>
                        </li>
                    ))}
                </ul>
                <p className="mt-4 font-semibold">
                    Total: ${cart.reduce((sum, item) => sum + item.price * item.quantity, 0).toFixed(2)}
                </p>
            </main>
        </div>
    );
}

export default App;

@import 'https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css';

Install: npm install @reduxjs/toolkit react-redux. Run: npm run dev. This manages cart state centrally.

Practical Applications

Redux simplifies state management in your e-commerce app, making it easier to scale features like user sessions and cart persistence.

Tips for Success

To master Redux:

  • Use Toolkit: Simplify setup with createSlice.
  • Keep State Minimal: Store only necessary data.
  • Debug with DevTools: Use Redux DevTools extension.
  • Test Actions: Verify state updates correctly.

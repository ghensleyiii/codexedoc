Branching and Merging

Welcome to Lesson 7 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn how to use Git branching and merging to work on features in parallel. By the end, you’ll be able to create branches, switch between them, and merge changes into your main codebase.

Introduction

Branching in Git is like a shepherd guiding sheep down separate paths to work on different tasks, then bringing them back together. Branches allow you to develop features or fixes without affecting the main codebase, and merging combines those changes. This lesson teaches you to use branches for organized, parallel development.

Key Concepts

This lesson covers essential branching and merging commands:

  • Creating Branches: Use `git branch feature` to create a new branch for a specific task.
  • Switching Branches: Use `git checkout feature` to work on a specific branch.
  • Merging: Use `git merge feature` to combine changes from one branch into another, typically `main`.

Code Example

Below is an example of creating a feature branch, adding a file, and merging it back into the main branch.

# Create and switch to a new branch
git branch feature
git checkout feature

# Add a new file and commit
echo "<h1>New Feature</h1>" > feature.html
git add feature.html
git commit -m "Add feature page"

# Switch back to main and merge
git checkout main
git merge feature

Practical Applications

Branching and merging are critical for team workflows in full-stack development. For example, when building a web app, one developer might work on a login feature on a separate branch while another updates the homepage. Merging combines these efforts seamlessly. Companies like GitHub and Atlassian rely on branching for efficient collaboration.

Tips for Success

Here are tips to master branching and merging:

  • Use descriptive branch names: E.g., `feature/login` or `fix/bug-123` for clarity.
  • Merge regularly: Keep branches up-to-date with `main` to avoid conflicts.
  • Test before merging: Ensure your feature works before combining it.

Assessment: Exam on SQL and Database Management

Testing Your SQL and Database Skills

You’ve completed Lessons 41–44, mastering relational databases, SQL, MongoDB, and integration. This exam tests your ability to build a secure order management API using PostgreSQL, MongoDB, and Node.js or Python, with optimized queries and tests.

Why This Assessment Matters

Order management APIs are common in e-commerce, requiring efficient database operations. This exam ensures you can build production-ready database-driven apps, a key full-stack skill.

Exam Overview

You’ll create a REST API for managing orders, using PostgreSQL for order data and MongoDB for logs, with Node.js (Sequelize, Mongoose) or Python (Django, PyMongo). Requirements include:

  • REST API: Build /api/orders for CRUD operations.
  • Databases: Use PostgreSQL for orders, MongoDB for logs.
  • Authentication: Secure endpoints with JWT.
  • Queries: Optimize SQL and MongoDB queries with indexes.
  • Testing: Write tests (Jest or Pytest) for API and database.

Sample Task: Building the Order API

Task: Create a REST API with CRUD endpoints for orders, using PostgreSQL and MongoDB, protected by JWT. Include a test for order creation and an optimized SQL query.

Sample Solution (Django-based):

# models.py (PostgreSQL)
from django.db import models

class Order(models.Model):
    customer_name = models.CharField(max_length=100)
    total = models.FloatField()
    created_at = models.DateTimeField(auto_now_add=True)

# serializers.py
from rest_framework import serializers
from .models import Order

class OrderSerializer(serializers.ModelSerializer):
    class Meta:
        model = Order
        fields = ['id', 'customer_name', 'total', 'created_at']

# views.py
from rest_framework import viewsets
from rest_framework.permissions import IsAuthenticated
from .models import Order
from .serializers import OrderSerializer

class OrderViewSet(viewsets.ModelViewSet):
    queryset = Order.objects.all()
    serializer_class = OrderSerializer
    permission_classes = [IsAuthenticated]

# log_orders.py (MongoDB)

# urls.py
from django.urls import path, include
from rest_framework.routers import DefaultRouter
from . import views

router = DefaultRouter()
router.register(r'orders', views.OrderViewSet)

urlpatterns = [
    path('api/', include(router.urls)),
]

# create_index.sql (PostgreSQL)
CREATE INDEX idx_order_created ON orders(created_at);

# optimized_query.sql
SELECT customer_name, total
FROM orders
WHERE created_at > '2025-01-01'
ORDER BY created_at DESC
LIMIT 10;

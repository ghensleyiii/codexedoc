Introduction to React

Introduction to React

Welcome to Lesson 41 of CODEXEDOC’s Full-Stack Development Course! React is a powerful JavaScript library for building dynamic user interfaces. In this lesson, you’ll learn React basics and create a simple component for your e-commerce app, transitioning from vanilla JavaScript.

What is React?

React is a component-based library for creating reusable UI elements. In 2025, it’s a leading choice for modern web apps due to its efficiency and ecosystem.

Why Use React?

React simplifies frontend development with reusable components and a virtual DOM, improving performance. For your e-commerce app, React enhances the cart and login UI.

Key React Concepts

Focus on these techniques:

  • Components: Build reusable UI blocks with JSX.
  • Props: Pass data to components.
  • State: Manage dynamic data with useState.
  • Setup: Use Vite for a modern React project.

Setting Up React

Follow these steps:

  1. Install Node.js: Ensure Node.js is installed.
  2. Create Project: Use Vite to set up a React app.
  3. Build Component: Create a product card component.
  4. Test Locally: Run the app with npm run dev.

Code Example: React Component

Here’s a simple React app with a product card:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>E-commerce React</title>
</head>
<body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
</body>
</html>

<!-- src/main.jsx -->
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')).render(
    <React.StrictMode>
        <App />
    </React.StrictMode>
);

<!-- src/App.jsx -->
import { useState } from 'react';

function App() {
    const [products] = useState([
        { id: 1, name: 'Laptop', price: 999.99 }
    ]);

    return (
        <div className="p-4">
            <h1 className="text-2xl font-bold">Store</h1>
            {products.map(product => (
                <div key={product.id} className="bg-white p-4 rounded shadow mt-4">
                    <h2 className="font-bold">{product.name}</h2>
                    <p>${product.price.toFixed(2)}</p>
                </div>
            ))}
        </div>
    );
}

export default App;

<!-- src/index.css -->
@import 'https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css';

Create a project: npm create vite@latest ecommerce -- --template react. Install dependencies: npm install. Run: npm run dev. Visit http://localhost:5173.

Practical Applications

React powers dynamic e-commerce frontends, replacing Lesson 31’s vanilla JavaScript. This component is the foundation for your app’s UI.

Tips for Success

To master React:

  • Use JSX: Write HTML-like code in JavaScript.
  • Keep Components Small: Break UI into reusable parts.
  • Learn Hooks: Start with useState and useEffect.
  • Test Locally: Use Vite’s hot reload for development.

Summary

React simplifies frontend development with components. By building a product card, you’ve started modernizing your e-commerce app. In the next lesson, you’ll build a full React frontend. Keep practicing React!

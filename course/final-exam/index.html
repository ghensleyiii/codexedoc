Earning Your Full-Stack Certification

You’ve completed Lessons 1–48, mastering full-stack development. This final exam tests your ability to build a secure booking system API and frontend, integrating all skills from HTML to deployment, to earn CODEXEDOC’s Full-Stack Development Certification.

Why This Exam Matters

Booking systems are common in industries like travel and events, requiring robust full-stack skills. This exam ensures you’re job-ready, validating your expertise for employers in 2025.

Exam Overview

You’ll create a REST API and frontend for a booking system, using React, Node.js/Express or Django, PostgreSQL for bookings, and MongoDB for logs. Requirements include:

  • Frontend: Build a React UI for browsing and booking.
  • REST API: Create endpoints for bookings and users.
  • Databases: Use PostgreSQL for bookings, MongoDB for logs.
  • Authentication: Secure endpoints with JWT.
  • Optimization: Cache queries, optimize SQL/MongoDB.
  • Testing: Write Jest or Pytest tests for API and frontend.
  • Deployment: Deploy to Heroku or AWS.

Sample Task: Building the Booking API

Task: Create a REST API with CRUD endpoints for bookings, using PostgreSQL and MongoDB, protected by JWT. Include a React frontend, optimized queries, and a test.

Sample Solution (Django-based):

# backend/models.py (PostgreSQL)
from django.db import models

class Booking(models.Model):
    user = models.ForeignKey('auth.User', on_delete=models.CASCADE)
    event = models.CharField(max_length=100)
    date = models.DateTimeField()
    status = models.CharField(max_length=20, default='pending')

# backend/serializers.py
from rest_framework import serializers
from .models import Booking

class BookingSerializer(serializers.ModelSerializer):
    class Meta:
        model = Booking
        fields = ['id', 'user', 'event', 'date', 'status']

# backend/views.py
from rest_framework import viewsets
from rest_framework.permissions import IsAuthenticated
from .models import Booking
from .serializers import BookingSerializer

class BookingViewSet(viewsets.ModelViewSet):
    queryset = Booking.objects.all()
    serializer_class = BookingSerializer
    permission_classes = [IsAuthenticated]

    def get_queryset(self):
        return self.queryset.filter(user=self.request.user)

# backend/urls.py
from django.urls import path, include
from rest_framework.routers import DefaultRouter
from . import views

router = DefaultRouter()
router.register(r'bookings', views.BookingViewSet)

urlpatterns = [
    path('api/', include(router.urls)),
]

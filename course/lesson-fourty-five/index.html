Real-Time Features with WebSockets

Welcome to Lesson 45 of CODEXEDOC’s Full-Stack Development Course! WebSockets enable real-time communication. In this lesson, you’ll add real-time product updates to your e-commerce app, notifying users when new products are added.

What are WebSockets?

WebSockets provide a persistent connection for bidirectional communication. In 2025, they’re ideal for real-time features like live notifications in web apps.

Why Use WebSockets?

WebSockets enhance your e-commerce app with instant updates, like new product alerts, improving user engagement over polling-based solutions.

Key WebSocket Concepts

Focus on these techniques:

  • Socket.IO: A library for WebSocket communication.
  • Events: Emit and listen for custom events.
  • Backend Integration: Notify clients on data changes.
  • Frontend Handling: Update UI in real-time.

Implementing WebSockets

Follow these steps:

  1. Install Socket.IO: Add to backend and frontend.
  2. Configure Backend: Emit events on product creation.
  3. Update Frontend: Listen for events in React.
  4. Test Real-Time: Verify live updates.

Code Example: WebSocket Integration

Here’s a backend and React frontend with real-time product updates:

<!-- server.js (Backend) -->
const express = require('express');
const mongoose = require('mongoose');
const cors = require('cors');
const http = require('http');
const { Server } = require('socket.io');
const app = express();
const server = http.createServer(app);
const io = new Server(server, {
    cors: {
        origin: 'http://localhost:5173',
        methods: ['GET', 'POST']
    }
});

app.use(cors());
app.use(express.json());

mongoose.connect('mongodb://localhost:27017/ecommerce', {
    useNewUrlParser: true,
    useUnifiedTopology: true
}).then(() => console.log('Connected to MongoDB'));

const productSchema = new mongoose.Schema({
    name: { type: String, required: true },
    price: { type: Number, required: true }
});

const Product = mongoose.model('Product', productSchema);

app.get('/api/products', async (req, res) => {
    const products = await Product.find();
    res.json(products);
});

app.post('/api/products', async (req, res) => {
    const product = new Product(req.body);
    await product.save();
    io.emit('newProduct', product);
    res.status(201).json(product);
});

server.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

<!-- src/App.jsx (Frontend) -->
import { useState, useEffect } from 'react';
import io from 'socket.io-client';

const socket = io('http://localhost:3000');

function App() {
    const [products, setProducts] = useState([]);
    const [error, setError] = useState(null);

    useEffect(() => {
        fetch('http://localhost:3000/api/products')
            .then(res => {
                if (!res.ok) throw new Error('Failed to fetch');
                return res.json();
            })
            .then(data => setProducts(data))
            .catch(err => setError(err.message));

        socket.on('newProduct', (product) => {
            setProducts(prev => [...prev, product]);
        });

        return () => socket.off('newProduct');
    }, []);

    return (
        <div className="p-4">
            <header className="bg-blue-600 text-white p-4">
                <h1 className="text-2xl font-bold">Store</h1>
            </header>
            <main className="container mx-auto">
                {error && <p className="text-red-600">Error: {error}</p>}
                <h2 className="text-xl font-bold my-4">Products</h2>
                <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
                    {products.map(product => (
                        <div key={product.id} className="bg-white p-4 rounded shadow">
                            <h3 className="font-bold">{product.name}</h3>
                            <p>${product.price.toFixed(2)}</p>
                        </div>
                    ))}
                </div>
            </main>
        </div>
    );
}

export default App;

Install backend: npm install express mongoose cors socket.io. Install frontend: npm install socket.io-client. Run backend: node server.js. Run frontend: npm run dev. Test by adding products via Postman.

Practical Applications

WebSockets enable live updates in your e-commerce app, like real-time product additions, enhancing user experience.

Tips for Success

To master WebSockets:

  • Use Socket.IO: Simplifies WebSocket setup.
  • Handle Disconnects: Reconnect gracefully.
  • Test Events: Verify real-time updates.
  • Secure Connections: Use HTTPS in production.

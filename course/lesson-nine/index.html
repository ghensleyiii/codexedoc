HTML Structure and Semantics

Welcome to Lesson 9 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn how to use HTML to create structured, accessible web pages. By the end, you’ll be able to build a semantic blog page using proper HTML tags.

Introduction

HTML is like a sheep pen, organizing content into clear, meaningful sections. Semantic HTML uses tags that describe their purpose, making pages accessible to users and search engines. This lesson covers the structure of HTML and semantic tags to create well-organized web content.

Key Concepts

This lesson covers HTML essentials:

  • Basic Structure: Use <!DOCTYPE html>, <html>, <head>, and <body> to set up a page.
  • Semantic Tags: Use <header>, <main>, <footer>, <article>, and <nav> to define content roles.
  • Accessibility: Semantic tags improve screen reader compatibility and SEO.

Code Example

Below is an example of a semantic HTML blog page, using tags to clearly define sections.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My Blog</title>
</head>
<body>
    <header>
        <h1>My Blog</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <article>
            <h2>Post Title</h2>
            <p>This is a blog post.</p>
        </article>
    </main>
    <footer>
        <p>© 2025 My Blog</p>
    </footer>
</body>
</html>

Practical Applications

Semantic HTML is the foundation of professional web development, used in sites like BBC or Medium. It ensures pages are accessible to screen readers, improves search engine rankings, and makes code easier to maintain. For example, a blog app uses <article> for posts and <nav> for menus, enhancing user experience.

Tips for Success

Here are tips to master HTML structure:

  • Use semantic tags: Choose <main> over <div> for clarity.
  • Validate your HTML: Use tools like the W3C validator to catch errors.
  • Think accessibility: Ensure every tag serves a purpose for all users.

Shearing Tip

Semantics shear away confusion for users and search engines, just as a shepherd organizes a flock for clarity. Semantic HTML makes your pages intuitive and inclusive.

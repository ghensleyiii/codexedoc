Local Storage

Welcome to Lesson 25 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn JavaScript local storage to persist data in the browser. By the end, you’ll be able to save and retrieve a user’s preferences.

Introduction

Local storage is like a shepherd’s journal, keeping records between visits. JavaScript’s localStorage API stores data persistently in the browser. This lesson teaches you to save and retrieve data for enhanced user experiences.

Key Concepts

This lesson covers local storage essentials:

  • localStorage: Use setItem and getItem to store and retrieve key-value pairs.
  • JSON: Use JSON.stringify and JSON.parse to store complex data.
  • Session Storage: Use sessionStorage for temporary, session-based data.

Code Example

Below is an example of saving and loading a user’s theme preference with local storage.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Theme Preference</title>
    <style>
        .dark { background: #333; color: #fff; }
        .light { background: #fff; color: #000; }
    </style>
</head>
<body>
    <select id="theme" onchange="saveTheme()">
        <option value="light">Light</option>
        <option value="dark">Dark</option>
    </select>
    <script>
        function saveTheme() {
            const theme = document.getElementById('theme').value;
            localStorage.setItem('theme', theme);
            applyTheme(theme);
        }
        function applyTheme(theme) {
            document.body.className = theme;
        }
        window.onload = () => {
            const savedTheme = localStorage.getItem('theme') || 'light';
            document.getElementById('theme').value = savedTheme;
            applyTheme(savedTheme);
        };
    </script>
</body>
</html>

Practical Applications

Local storage is used in apps like Todoist or Medium to save user settings or drafts. For example, a note-taking app uses local storage to persist unsaved notes. These skills are essential for full-stack developers to enhance user retention.

Tips for Success

Here are tips to master local storage:

  • Store strings only: Use JSON for arrays or objects in localStorage.
  • Check storage limits: Browsers typically allow 5-10MB for local storage.
  • Clear unused data: Use removeItem or clear to manage storage.

Shearing Tip

Local storage shears away data loss with persistent records, just as a shepherd’s journal preserves flock details. It keeps user data safe across sessions.

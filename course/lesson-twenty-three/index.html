Mastering TypeScript Basics

Welcome to Lesson 23 of CODEXEDOC’s Full-Stack Development Course! TypeScript is a superset of JavaScript that adds static types, improving code reliability. In this lesson, you’ll learn TypeScript basics, including types, interfaces, and type annotations.

What is TypeScript?

TypeScript, developed by Microsoft in 2012, compiles to JavaScript and adds type safety. It’s widely used in 2025 for large-scale apps, especially with frameworks like React, to catch errors early.

Key TypeScript Concepts

Focus on these features:

  • Types: Define variable types (e.g., number, string).
  • Interfaces: Define object shapes (e.g., interface User).
  • Type Annotations: Specify types for variables/functions (e.g., let age: number).
  • Union Types: Allow multiple types (e.g., string | number).

Code Example: TypeScript Basics

Here’s a TypeScript example:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>TypeScript Example</title>
</head>
<body>
    <script src="script.js"></script>
</body>
</html>

<!-- script.ts -->
interface User {
    name: string;
    age: number;
}
const user: User = { name: "Alice", age: 25 };
function greet(person: string): string {
    return `Hello, ${person}!`;
}
console.log(greet(user.name)); // Output: Hello, Alice!

Compile script.ts to script.js. This code uses interfaces and type annotations.

Practical Applications

TypeScript ensures type safety in React apps. In your task manager project, you’ll use TypeScript to define task data structures, reducing runtime errors.

Tips for Success

To master TypeScript:

  • Start with Types: Add types to variables and functions gradually.
  • Use Interfaces: Define reusable object shapes.
  • Install TypeScript: Use npm install -g typescript and tsc to compile.
  • Check Errors: Use VS Code for real-time type checking.

Summary

TypeScript adds type safety to JavaScript, improving code reliability. By mastering types and interfaces, you’re ready for typed apps. In the next lesson, you’ll learn React for building UI components. Keep practicing TypeScript!

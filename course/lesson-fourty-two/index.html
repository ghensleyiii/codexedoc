Building a React Frontend

Welcome to Lesson 42 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll build a React frontend for your e-commerce app, replacing Lesson 31’s vanilla JavaScript with components for products, cart, and login, integrated with Lesson 40’s backend.

What is a React Frontend?

A React frontend uses components to create a dynamic, modular UI. It communicates with a backend API to display and manage data, like products and user sessions.

Why Build with React?

React improves scalability and maintainability over vanilla JavaScript. For your e-commerce app, it streamlines cart management and user authentication.

Key Frontend Concepts

Focus on these techniques:

  • Component Hierarchy: Organize UI into parent-child components.
  • API Calls: Fetch data with useEffect.
  • Conditional Rendering: Show/hide UI based on state.
  • Event Handling: Manage user interactions like clicks.

Building the Frontend

Follow these steps:

  1. Create Components: Build product, cart, and login components.
  2. Fetch Data: Connect to the Lesson 40 backend.
  3. Manage State: Use useState for cart and login.
  4. Test Locally: Run with npm run dev.

Code Example: React Frontend

Here’s a React frontend with product and cart components:

<!-- src/App.jsx -->
import { useState, useEffect } from 'react';

function App() {
    const [products, setProducts] = useState([]);
    const [cart, setCart] = useState([]);
    const [error, setError] = useState(null);

    useEffect(() => {
        fetch('http://localhost:3000/api/products', {
            headers: { 'Content-Type': 'application/json' }
        })
            .then(res => {
                if (!res.ok) throw new Error('Failed to fetch');
                return res.json();
            })
            .then(data => setProducts(data))
            .catch(err => setError(err.message));
    }, []);

    const addToCart = (product) => {
        setCart(prev => {
            const item = prev.find(item => item.id === product.id);
            if (item) {
                return prev.map(item =>
                    item.id === product.id ? { ...item, quantity: item.quantity + 1 } : item
                );
            }
            return [...prev, { ...product, quantity: 1 }];
        });
    };

    const removeFromCart = (id) => {
        setCart(prev => prev.filter(item => item.id !== id));
    };

    return (
        <div className="p-4">
            <header className="bg-blue-600 text-white p-4">
                <h1 className="text-2xl font-bold">Store</h1>
            </header>
            <main className="container mx-auto">
                {error && <p className="text-red-600">Error: {error}</p>}
                <h2 className="text-xl font-bold my-4">Products</h2>
                <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
                    {products.map(product => (
                        <div key={product.id} className="bg-white p-4 rounded shadow">
                            <h3 className="font-bold">{product.name}</h3>
                            <p>${product.price.toFixed(2)}</p>
                            <button
                                className="bg-blue-600 text-white px-4 py-2 rounded"
                                onClick={() => addToCart(product)}
                            >
                                Add to Cart
                            </button>
                        </div>
                    ))}
                </div>
                <h2 className="text-xl font-bold my-4">Cart</h2>
                <ul className="bg-white p-4 rounded shadow">
                    {cart.map(item => (
                        <li key={item.id} className="flex justify-between py-2">
                            <span>{item.name} (x{item.quantity}) - ${(item.price * item.quantity).toFixed(2)}</span>
                            <button
                                className="text-red-600"
                                onClick={() => removeFromCart(item.id)}
                            >
                                Remove
                            </button>
                        </li>
                    ))}
                </ul>
                <p className="mt-4 font-semibold">
                    Total: ${cart.reduce((sum, item) => sum + item.price * item.quantity, 0).toFixed(2)}
                </p>
            </main>
        </div>
    );
}

export default App;

Tips for Success

To master React frontends:

  • Organize Components: Keep logic modular.
  • Handle Errors: Display fetch errors to users.
  • Use Tailwind: Style consistently with prior lessons.
  • Test APIs: Verify backend connectivity.

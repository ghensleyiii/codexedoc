Database Integration with Python

Mastering Database Integration

Welcome to Lesson 39! Databases store app data persistently. In this lesson, you’ll learn to integrate SQL databases (e.g., PostgreSQL) with Python using Django’s ORM and SQLAlchemy for Flask.

What is Database Integration?

Database integration connects apps to relational databases for data management. In 2025, PostgreSQL is a popular choice for Python apps, offering robust features for Django and Flask.

Key Concepts

Focus on these techniques:

  • Django ORM: Query databases with model-based syntax.
  • SQLAlchemy: Use for Flask database operations.
  • Migrations: Manage schema changes.
  • Connections: Configure database URLs securely.

Code Example: Database Integration

Here’s Django and Flask database integration:

# Django: models.py
from django.db import models

class User(models.Model):
    username = models.CharField(max_length=50)
    email = models.EmailField(unique=True)

# Django: views.py
from django.shortcuts import render
from .models import User

def user_list(request):
    users = User.objects.all()
    return render(request, 'users/user_list.html', {'users': users})

# Flask: app.py
from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'postgresql://user:pass@localhost/db'
db = SQLAlchemy(app)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(50), unique=True)

@app.route('/users')
def get_users():
    users = User.query.all()
    return {'users': [{'username': user.username} for user in users]}

if __name__ == '__main__':
    app.run(port=5000)

Run Django with python manage.py runserver or Flask with python app.py.

Practical Applications

Databases store user data for apps. In your social media dashboard project, you’ll use Django’s ORM for posts and users.

Tips for Success

To master database integration:

  • Use PostgreSQL: Install and configure locally.
  • Run Migrations: Use python manage.py makemigrations for Django.
  • Secure Credentials: Store database URLs in environment variables.
  • Test Queries: Verify data retrieval locally.

Summary

Database integration enables persistent data storage. By mastering Django’s ORM and SQLAlchemy, you’re ready for data-driven apps. Next, you’ll learn Pytest for testing. Keep integrating databases!

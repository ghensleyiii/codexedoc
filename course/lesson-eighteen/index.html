JavaScript Fundamentals

Welcome to Lesson 18 of CODEXEDOC’s Full-Stack Development Course! JavaScript powers dynamic web apps. In this lesson, you’ll learn the basics to start building your Task Manager App, with applications to e-commerce.

What is JavaScript?

JavaScript is a versatile language for adding interactivity to websites, essential for modern web development in 2025.

Why Learn JavaScript?

JavaScript enables task creation in your Task Manager App or cart updates in an e-commerce store, making apps dynamic.

Key JavaScript Concepts

Focus on these fundamentals:

  • Variables: Declare with let, const, or var.
  • Data Types: Numbers, strings, arrays, objects.
  • Functions: Define reusable code blocks.
  • Conditionals: Use if and switch for logic.

Getting Started

Follow these steps:

  1. Add JavaScript: Include a script in HTML.
  2. Store Tasks: Use arrays for task data.
  3. Create Functions: Add tasks dynamically.
  4. Test in Browser: Use console to debug.

Code Example: Basic Task List

Here’s a simple task list for your app:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Task Manager</title>
    <link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
</head>
<body class="p-4">
    <h1 class="text-2xl font-bold">Task Manager</h1>
    <input id="taskInput" class="border p-2" placeholder="Add task">
    <button onclick="addTask()" class="bg-blue-600 text-white px-4 py-2 rounded">Add</button>
    <ul id="taskList" class="mt-4"></ul>
    <script src="script.js"></script>
</body>
</html>

<!-- script.js -->
const tasks = [];

function addTask() {
    const input = document.getElementById('taskInput');
    const task = input.value.trim();
    if (task) {
        tasks.push(task);
        input.value = '';
        displayTasks();
    }
}

function displayTasks() {
    const taskList = document.getElementById('taskList');
    taskList.innerHTML = '';
    tasks.forEach(task => {
        const li = document.createElement('li');
        li.textContent = task;
        li.className = 'py-2';
        taskList.appendChild(li);
    });
}

Save files, open index.html in a browser, and test adding tasks. This logic applies to e-commerce product lists.

Practical Applications

JavaScript fundamentals enable task management or product catalog updates, forming the core of interactive web apps.

Tips for Success

To master JavaScript basics:

  • Prefer const: Use for non-reassigned variables.
  • Debug with Console: Log values with console.log.
  • Practice Arrays: Store tasks or products.
  • Test Incrementally: Build and test small pieces.

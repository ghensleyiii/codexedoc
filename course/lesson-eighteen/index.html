Introduction to Node.js

Welcome to Lesson 18! You’ll learn Node.js for server-side programming. By the end, you’ll build a “Hello, World!” server.

Introduction

Node.js is like a shepherd running JavaScript outside the browser, enabling server-side apps. This lesson introduces Node.js, its HTTP module, and npm for package management.

Key Concepts

Focus areas include:

  • Node.js: JavaScript runtime for servers.
  • HTTP Module: Create basic servers.
  • npm: Manage packages and dependencies.

Code Example

Here’s a simple Node.js server.

const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello, World!\n');
});

server.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

Save as server.js, run with node server.js, and visit http://localhost:3000.

Practical Applications

Node.js powers apps like Netflix for scalable back-ends. It’s used to serve APIs or handle real-time data. These skills are critical for back-end development roles.

Tips for Success

Master Node.js with these tips:

  • Verify Node.js installation with node --version.
  • Use npm to install packages efficiently.
  • Test servers locally before deployment.

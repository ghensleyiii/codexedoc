Testing Your JavaScript and React Proficiency

You’ve completed the JavaScript section (Lessons 18–26) of CODEXEDOC’s Full-Stack Development Course, covering JavaScript fundamentals, ES6+, DOM manipulation, asynchronous JavaScript, functional programming, TypeScript, React, state management, and testing. This practical exam tests your ability to build a dynamic product filter component, a common feature in e-commerce apps, using React, TypeScript, and related skills.

Why This Assessment Matters

Dynamic filtering is a key feature in modern web apps, requiring proficiency in React state, TypeScript, and JavaScript. This exam simulates a real-world task, ensuring you can build interactive, type-safe components, a critical skill for front-end developers.

Exam Overview

In this practical exam, you’ll create a product filter component that displays a list of products and allows filtering by category (e.g., “Electronics,” “Clothing”) using React, TypeScript, and Context API. Requirements include:

  • React Components: Build a ProductList and FilterBar component.
  • TypeScript: Use interfaces for product data (e.g., interface Product).
  • Context API: Manage filter state globally.
  • Functional Programming: Use filter to update the product list.
  • Asynchronous JavaScript: Fetch products from a mock API using async/await.
  • Testing: Write Jest tests for filtering functionality.
  • Styling: Use Tailwind CSS or SCSS for a responsive UI.
  • Accessibility: Ensure keyboard navigation and ARIA attributes.

Sample Task: Building the Product Filter

Task: Create a product filter component that fetches products from a mock API, displays them, and filters by category. Include a Jest test for filtering.

Sample Solution:

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Product Filter</title>
</head>
<body>
    <div id="root"></div>
    <script src="app.js"></script>
</body>
</html>

<!-- App.tsx -->
import React, { createContext, useContext, useState, useEffect } from 'react';
import ReactDOM from 'react-dom';

interface Product {
    id: number;
    name: string;
    category: string;
}

interface FilterContextType {
    category: string;
    setCategory: (category: string) => void;
    products: Product[];
}

const FilterContext = createContext({ category: '', setCategory: () => {}, products: [] });

const App: React.FC = () => {
    const [products, setProducts] = useState([]);
    const [category, setCategory] = useState('');

    useEffect(() => {
        const fetchProducts = async () => {
            try {
                const response = await fetch('products.json');
                const data = await response.json();
                setProducts(data);
            } catch (error) {
                console.error('Error fetching products');
            }
        };
        fetchProducts();
    }, []);

    return (
        <FilterContext.Provider value={{ category, setCategory, products }}>
            <FilterBar />
            <ProductList />
        </FilterContext.Provider>
    );
};

const FilterBar: React.FC = () => {
    const { setCategory } = useContext(FilterContext);
    return (
        <div>
            <select onChange={(e) => setCategory(e.target.value)} aria-label="Filter by category">
                <option value="">All</option>
                <option value="Electronics">Electronics</option>
                <option value="Clothing">Clothing</option>
            </select>
        </div>
    );
};

const ProductList: React.FC = () => {
    const { products, category } = useContext(FilterContext);
    const filteredProducts = category
        ? products.filter(product => product.category === category)
        : products;
    return (
        <ul>
            {filteredProducts.map(product => (
                <li key={product.id}>{product.name}</li>
            ))}
        </ul>
    );
};

ReactDOM.render(<App />, document.getElementById('root'));

<!-- App.test.tsx -->
import { render, screen, fireEvent } from '@testing-library/react';
import App from './App';

describe('Product Filter', () => {
    test('filters products by category', () => {
        render(<App />);
        const select = screen.getByLabelText('Filter by category');
        fireEvent.change(select, { target: { value: 'Electronics' } });
        expect(screen.getByText('Laptop')).toBeInTheDocument();
        expect(screen.queryByText('Shirt')).not.toBeInTheDocument();
    });
});

This solution uses React, TypeScript, Context API, async/await, and Jest, with Tailwind CSS or SCSS for styling (not shown). Products are fetched from a mock API, filtered functionally, and tested.

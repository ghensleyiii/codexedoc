CSS Basics and Styling

Mastering CSS Basics and Styling

Welcome to Lesson 11 of CODEXEDOC’s Full-Stack Development Course! CSS (Cascading Style Sheets) brings your HTML to life by adding colors, fonts, and layouts. In this lesson, we’ll cover CSS fundamentals, including selectors, properties, and how to apply styles to your web pages. By the end, you’ll be able to style a basic webpage with confidence.

What is CSS?

CSS is a stylesheet language used to control the appearance of HTML elements. Introduced in 1996 by the W3C, CSS allows you to separate content (HTML) from presentation (styles), making websites visually appealing and maintainable. In 2025, CSS is essential for creating modern, responsive web designs.

Why Use CSS?

CSS enhances user experience by making websites attractive and consistent. It enables you to apply styles across multiple pages, adjust layouts for different devices, and maintain a cohesive design. Mastering CSS is key to building professional websites and preparing for frameworks like Bootstrap.

Key CSS Concepts

Here are the core components of CSS:

  • Selector: Targets HTML elements (e.g., h1, .class, #id).
  • Property: Defines what to style (e.g., color, font-size).
  • Value: Specifies the style (e.g., blue, 16px).
  • Cascade: Determines which styles apply when multiple rules target the same element, based on specificity and order.

Applying CSS

You can apply CSS in three ways (learn more at 

w3.org
):

  1. Inline CSS: Add styles directly in HTML using the style attribute (e.g., <p style="color: blue;">).
  2. Internal CSS: Place styles in a <style> tag in the HTML <head>.
  3. External CSS: Link a separate .css file using <link rel="stylesheet" href="styles.css">.

Code Example: Basic CSS Styling

Here’s an HTML page with external CSS:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styled Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Welcome</h1>
    </header>
    <main>
        <p>This is a styled paragraph.</p>
    </main>
</body>
</html>


h1 {
    color: navy;
    font-family: 'Roboto Mono', monospace;
}
p {
    font-size: 16px;
    color: #333;
}

This example styles a heading and paragraph using an external CSS file, linked in the HTML <head>.

Practical Applications

CSS is used to style every modern website, from personal blogs to e-commerce platforms. For example, you’ll use CSS to style buttons, navigation menus, and text in your portfolio project. In this course, you’ll apply CSS to create visually appealing layouts for your e-commerce product page.

Tips for Success

To write effective CSS:

  • Use External CSS: Keep styles in a separate file for maintainability.
  • Test in Browser: Use browser developer tools (F12) to experiment with styles live.
  • Organize Selectors: Group related styles to keep your CSS clean.
  • Learn Specificity: Understand how CSS prioritizes rules to avoid unexpected styles.

Summary

CSS is the key to making your HTML visually engaging. By mastering selectors, properties, and styling methods, you can create beautiful web pages. In the next lesson, you’ll dive into the CSS box model and layouts, learning how to control the structure of your designs. Keep practicing CSS to build your styling skills!

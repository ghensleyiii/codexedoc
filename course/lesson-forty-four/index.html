Database Management
Integrating Databases with Node.js and Python

Mastering Database Integration

Welcome to Lesson 44! Integrating databases with apps enables persistent storage. In this lesson, you’ll learn to connect PostgreSQL and MongoDB to Node.js (using Sequelize, Mongoose) and Python (using Django ORM, PyMongo), building on prior backend skills.

What is Database Integration?

Integration connects app logic to databases for CRUD operations. In 2025, Node.js and Python commonly use PostgreSQL for relational data and MongoDB for NoSQL, powering REST APIs.

Key Concepts

Focus on these techniques:

  • Node.js with Sequelize: ORM for PostgreSQL.
  • Node.js with Mongoose: ODM for MongoDB.
  • Python with Django ORM: Manage PostgreSQL.
  • Python with PyMongo: Query MongoDB.

Code Example: Database Integration

Here’s Node.js and Python integration:

// Node.js: Sequelize with PostgreSQL
const { Sequelize, DataTypes } = require('sequelize');
const sequelize = new Sequelize('postgresql://user:pass@localhost/db');

const Product = sequelize.define('Product', {
    name: { type: DataTypes.STRING, allowNull: false },
    price: { type: DataTypes.FLOAT, allowNull: false }
});

async function getProducts() {
    await sequelize.sync();
    return await Product.findAll();
}

// Node.js: Mongoose with MongoDB
const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost/db');

const productSchema = new mongoose.Schema({
    name: String,
    price: Number
});
const Product = mongoose.model('Product', productSchema);

async function getMongoProducts() {
    return await Product.find();
}

# Python: Django ORM with PostgreSQL
from django.db import models

class Product(models.Model):
    name = models.CharField(max_length=100)
    price = models.FloatField()

def get_products():
    return Product.objects.all()

# Python: PyMongo with MongoDB
from pymongo import MongoClient

client = MongoClient('mongodb://localhost/db')
db = client['db']

def get_mongo_products():
    return db.products.find()

Run these to query products from PostgreSQL and MongoDB.

Practical Applications

Integration powers apps like inventory systems. In your project, you’ll use Node.js or Python to connect to PostgreSQL and MongoDB for product and transaction data.

Tips for Success

To master integration:

  • Secure Connections: Use environment variables for credentials.
  • Use ORMs/ODMs: Simplify queries with Sequelize or Django ORM.
  • Test Locally: Verify connections with sample data.
  • Handle Errors: Catch database connection failures.

Summary

Database integration enables robust apps. By mastering Node.js and Python connections, you’re ready for data-driven projects. Next, you’ll build an inventory management system. Keep integrating databases!

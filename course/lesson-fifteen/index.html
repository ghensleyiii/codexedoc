DOM Manipulation

Welcome to Lesson 15 of CODEXEDOC’s Full-Stack Development Course! In this lesson, you’ll learn how to manipulate the Document Object Model (DOM) with JavaScript to dynamically update web pages. By the end, you’ll be able to change text and styles on button clicks.

Introduction

The DOM is like a shepherd’s map of the flock, representing a web page’s structure as a tree of elements. JavaScript lets you modify this tree to update content or styles dynamically. This lesson teaches you to select and change elements for interactive pages.

Key Concepts

This lesson covers DOM manipulation essentials:

  • Selecting Elements: Use document.getElementById or querySelector to target elements.
  • Modifying Content: Change text with textContent or innerHTML.
  • Updating Styles: Modify CSS via the style property.

Code Example

Below is an example of changing text and background color when a button is clicked.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>DOM Manipulation</title>
</head>
<body>
    <h1 id="title">Hello</h1>
    <button onclick="update()">Update</button>
    <script>
        function update() {
            const title = document.getElementById('title');
            title.textContent = 'Updated!';
            title.style.backgroundColor = 'lightblue';
        }
    </script>
</body>
</html>

Practical Applications

DOM manipulation is used in apps like Twitter or Instagram to update feeds or toggle themes. For example, a portfolio site might use JavaScript to highlight project cards on hover. These skills are vital for full-stack developers to create dynamic front-end features.

Tips for Success

Here are tips to master DOM manipulation:

  • Use specific selectors: Prefer getElementById for single elements to improve performance.
  • Avoid innerHTML for security: Use textContent unless adding HTML is necessary.
  • Test changes: Use browser dev tools to inspect DOM updates.

Shearing Tip

DOM manipulation shears away static content with dynamic updates, just as a shepherd adjusts a flock’s path. Selecting and modifying elements brings your pages to life.

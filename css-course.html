1. Introduction to CSS
p { color: blue; }

CSS (Cascading Style Sheets) styles HTML elements:

  • Selector: Targets elements (e.g., p).
  • Property: Defines what to style (e.g., color).
  • Value: Sets the style (e.g., blue).

Try it: Add this to a <style> tag in an HTML file!

2. Inline CSS
<p style="color: red;">Text</p>

Inline CSS applies styles directly to elements:

  • style: Attribute for inline styling.

Try it: Add this to an HTML element!

3. Internal CSS
<style> p { font-size: 16px; } </style>

Internal CSS uses a <style> tag in the <head>:

  • Styles multiple elements at once.

Try it: Add this to your HTML <head>!

4. External CSS
<link rel="stylesheet" href="styles.css">

External CSS links a separate file:

  • href: Path to the CSS file.

Try it: Create a styles.css file and link it!

5. CSS Selectors
.class { color: green; } #id { font-weight: bold; }

Selectors target elements:

  • .class: Targets classes.
  • #id: Targets IDs.

Try it: Use these with HTML elements!

6. Colors in CSS
p { color: #ff0000; background-color: rgb(0, 255, 0); }

Colors can be set using names, hex, or RGB:

  • #ff0000: Hex for red.
  • rgb(0, 255, 0): Green in RGB.

Try it: Change some colors!

7. Fonts
p { font-family: Arial; font-size: 20px; }

Style text with fonts:

  • font-family: Sets the font.
  • font-size: Sets the size.

Try it: Customize text!

8. Text Alignment
p { text-align: center; }

Align text:

  • text-align: Left, right, center, or justify.

Try it: Align some text!

9. Margins
div { margin: 10px; }

Margins create space outside elements:

  • margin: Sets all sides.

Try it: Add margins to a div!

10. Padding
div { padding: 15px; }

Padding creates space inside elements:

  • padding: Sets all sides.

Try it: Add padding to a div!

30. CSS Transitions
button { transition: background-color 0.3s; } button:hover { background-color: blue; }

Transitions animate property changes:

  • transition: Property and duration.

Try it: Add a hover transition!
